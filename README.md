# SoulMates 
A dating site which will help you find your soulmate today!

# Technologies
### Frontend
* HTML
* CSS
* Vanilla JS (Fetch API & Ajax)
### Backend
* Spring boot Framework
* Spring security

## Testing tools
* Junit5
* Mockito 

## Database
* Mysql
* JDBC session Managment
